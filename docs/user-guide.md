# Руководство пользователя

## Первый запуск

NetBypass запускается с правами администратора, потому что обычный пользователь
Windows не может изменять `C:\Windows\System32\drivers\etc\hosts`.

При первом открытии:

1. Перейдите в раздел **Сервисы**.
2. Оставьте включёнными только необходимые сервисы.
3. Нажмите **Сохранить и применить**.
4. Убедитесь, что на главной появился статус **Защита активна**.

После этого приложение можно закрыть. Оно не работает в фоне и не требует
автозапуска: записи находятся непосредственно в системном `hosts`.

## Главная страница

Большая круглая кнопка управляет текущим состоянием:

- **Включить** — записывает выбранные модули;
- **Отключить** — удаляет управляемый блок NetBypass.

Возможные статусы:

- **Не настроено** — записей NetBypass в файле нет;
- **Защита активна** — активные записи совпадают с выбранными сервисами;
- **Требуется применить изменения** — выбор сервисов отличается от записанного;
- **Файл hosts требует внимания** — маркеры блока повреждены или изменены.

## Изменение списка сервисов

Изменение галочек само по себе не перезаписывает системный файл. После выбора
нажмите **Сохранить и применить**. Это позволяет спокойно настроить список и
внести изменения одной операцией.

## Отключение и восстановление

Обычное отключение удаляет блок между маркерами:

```text
# NETBYPASS-BEGIN
...
# NETBYPASS-END
```

Кнопка **Восстановить hosts** предназначена для проблемных случаев. После
подтверждения она удаляет известные записи NetBypass, не заменяя файл
заводским шаблоном и не удаляя чужие строки.

Резервная копия, созданная перед первой записью, хранится в:

```text
%LOCALAPPDATA%\NetBypass\Backups\hosts.initial.bak
```
